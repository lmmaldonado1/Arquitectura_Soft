=== ENGLISH ===
1 - Copy or move the PAGE_WEB folder to your local or remote XAMPP server, LARAGON

2 - Create a database in MYSQL with the name linux_master, 
select it and import the system database with phpmyadmin or another graphical MYSQL
administrator that you use, the file is called linux_master.sql and is in the DB folder.

3 – Open the config.php file with your favorite code editor and set it to server data only.
 The file is in the "model" folder.

Default user database:
USER: admin
PASS: Root

Default users to enter the WEBSITE:

Admin:
USER: admin
PASS: admin

Customer:
USER: client
PASS: client
